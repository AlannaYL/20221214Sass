01.Sass 介紹

Sass 是 CSS 的預處理器，目的在減少CSS重工

Sass 有變數、函式、流程控制、嵌套(巢狀寫法)、繼承、mixins 這些功能，讓我們使用，比 CSS 在寫樣式時更有彈性

Sass 可以更有效率維護樣式的結構

Sass 需要編譯成原生的 CSS 才能使用，瀏覽器看不懂 Sass

02.Sass 的資料型態

  1. 1.number(數字): 數字氛圍有單位跟無單位，1, 10, 100, 1px, 2rem等
  2. 2.String(字串): 有引號跟無引號
  3. 3.color(顏色): blue, #007bff, rgba(0, 0, 0, 0.1)
  4. 4.boolean
  5. 5.null(空值)
  6. 6.list(向陣列: 半形空格或逗號分隔，例如 20px 0 20px，'Courier New', Courier, monospace
  7. 7.map

03. 變數

宣告使用 $，變數位置即是作用域，就像 JS 的 let 宣告，寫在大括號裡面市區域變數，最外面式全域變數

變數底線 _ 和 - 兩者是一樣的變數，例如 $bg-dark、$bg-dark

04. 流程控制

@if、@else if、@else

判斷條件可以小括號，也可以不括號

使用 and、or、not 出裡判斷運算式的結果是 true 或 false


啾啾

@for 迴圈有兩種

@for $i from (start number) through (end number) {...} 包含 end，結束 number

@for $i from (start number) to (end number) {...} 不包含 end

05. 巢狀(嵌套，nested)

& 佔位符號可以替換父選擇器

勿過度使用，考慮用最少的選擇器，選擇到想要的位置，盡量在3~4層

過度使用嵌套，編譯後選擇器會過長，不易維護、閱讀，造成CSS檔案較大
